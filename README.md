# Jetpack Compose example

this repository is an implementation of Jetpack Compose with a search of products using the MercadoLibre Api and an product detail Screen

## Requirements
- Android Studio 2020.3.1 Patch 2
- Kotlin 1.5.21
- Gradle 7.0.2
- Java JDK 8
- Git
